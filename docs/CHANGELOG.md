# Changelog

## 1.1.0 - 2026-06-29

- Novo dashboard financeiro com oito indicadores operacionais.
- Filtros Hoje, Ontem, Semana, Mês, Ano e período personalizado.
- Métricas consolidadas em uma única consulta SQLite.
- Inclusão de status no cadastro e edição de vendas.
- Atualização automática dos indicadores após alterações.
- Migrações SQLite e Supabase para status de vendas.

## 1.0.0 - 2026-06-28

- Reorganização completa em `src/frontend`, `src/backend` e `src/database`.
- Migração do servidor HTTP nativo para Express.
- Separação em routes, controllers, services e repositories.
- Preservação da API e dos dados SQLite existentes.
- Inclusão dos esquemas SQLite e PostgreSQL/Supabase.
- Organização de logs e backups.
- Criação da documentação técnica do projeto.
