# Requisitos

## Ambiente

- Windows, Linux ou macOS.
- Node.js 22.5 ou superior, necessário para `node:sqlite`.
- Navegador moderno com suporte a `fetch`.

## Dependências

- Express 5 para HTTP e arquivos estáticos.
- `pg` reservado para a futura integração PostgreSQL/Supabase.

## Configuração

Copie as variáveis documentadas em `.env.example` para `.env`. Nunca publique o `.env`.
