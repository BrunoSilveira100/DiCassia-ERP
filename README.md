---

# 📖 Sobre o Projeto

O **DiCassia ERP** é um sistema web criado para facilitar a gestão diária de uma loja de perfumaria, cosméticos e acessórios.

O objetivo é oferecer uma plataforma simples, rápida e intuitiva para controlar toda a operação da loja em um único lugar.

Entre os principais recursos estão:

- 💰 Controle de Vendas
- 📦 Controle de Estoque
- 👤 Cadastro de Clientes
- 🏢 Cadastro de Fornecedores
- 💳 Contas a Pagar
- 📊 Dashboard Financeiro
- 📈 Fluxo de Caixa
- 📑 Relatórios
- 🔒 Login Seguro
- ☁️ Banco de Dados Online

## Nova Venda

Cadastro rápido de vendas contendo:

- Data
- Cliente
- Produto
- Quantidade
- Forma de pagamento
- Valor recebido
- Status
- Observações

Ao concluir uma venda o sistema:

- Atualiza o estoque
- Atualiza o dashboard
- Atualiza o fluxo de caixa
- Salva no banco de dados
